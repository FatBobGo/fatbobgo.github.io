hexo blog设置图片大小的方法

有时候在hexo博客插入图片的时候，是要调整图片大小以获得更好的阅读体验。

使用markdown语法和html img语法

source: hexo设置图片大小

在hexo中一般插入图片的方式为：

!(图片名称)[图片地址]

如果想要控制图片大小，可以用html的方式来插入图片

<img src="图片地址" width="50%" height="50%" style="margin: 0 auto;"/>

其中后面的style="margin:0 auto;"用来控制图片的居中，图片大小有width和height控制，可以输入50%这样的比例，也可以直接输入大小200px这样的值

使用别人写的插件/扩展

source: Hexo 中扩展 Markdown 语法设置图片的大小

下载js

把github js代码拷贝到本地博客目录.\themes\Chic\source\js\hexo_resize_image.js。

引用js

在\themes\Chic\layout\layout.ejs里面的</body>前面添加一句<script src="<%- config.root %>js/hexo_resize_image.js"></script>就可以了。
