N1配置OpenClash

Author: Fat Bob Date: December 24, 2020  19:50:40 Category: programming

今天折腾了一个早上终于把OpenClash配置好了。开心。😊

不知道OpenClash为什么设计得这么复杂！这次的成功全靠功能一键生成配置文件。

网络环境

N1作为旁路由连接主路由上网。旁路由里面安装了AdGuard和OpenClash。

大概步骤

这里简单罗列一下大概步骤。搞清楚了大概的设置顺序之后参考下面的文章就可以设置成功了。

  1. 服务器与策略组管理 -> 服务器节点配置 添加自己的服务器先。
  2. 全局设置 -> 模式设置：
    设置Fake-IP模式
    代理模式选择 Rule【策略代理】
    勾选 仅代理命中规则流量
    保存配置
  3. 全局设置 -> 常规设置：
    选择自己的内核。我使用N1所以选择 linux-armv8
  4. 全局设置 -> DNS设置：
    勾选 本地DNS劫持
    勾选 自定义上游DNS服务器。然后选择1-2个DNS服务器
  5. 全局设置 -> 版本更新：
    基本上就是把所有都更新一遍。这里可能需要你使用Passwall之类连上自己的梯子。
  6. 全局设置 -> 服务器与策略组管理
    勾选 一键生成配置文件
  7. 点击 应用配置
    因为刚刚勾选了 一键生成配置文件，这样OpenClash就会自动的生成一个配置文件 config.yaml

稍等一下，这样在运行状态下如果看到都是绿色的话，说明可以科学上网了。
