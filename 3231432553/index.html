ERROR Plugin load failed: hexo-filter-github-emojis

Author: Fat Bob Date: December 24, 2020  0:26:56

背景

今天打开博客准备写文章的时候发现在新建文章和生成文章的时候报错：

ERROR Plugin load failed: hexo-filter-github-emojis

修复

通过重新安装插件来修复问题

npm install hexo-filter-github-emojis --save
