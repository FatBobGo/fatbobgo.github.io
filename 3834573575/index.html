Visual Studio Code Can't Run in Ubuntu

Author: Fat Bob Date: February 5, 2019  20:27:09 Category: software;

在Ubuntu 18.04下安装好微软的Visual Studio Code之后，发现运行之后没有反应：应用程序没有加载出来。通过搜索发现运来是权限的问题。

Root Cause

vscode的配置文件被加上了root权限

解决方法

把配置文件的root权限去掉

因为是sudo，所有需要输入密码。
