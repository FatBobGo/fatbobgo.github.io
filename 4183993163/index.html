Ubuntu 19.10中vscode无法切换中文

Author: Fat Bob Date: February 9, 2020  19:33:32 Category: programming

在商城安装VS Code之后，发现无法输入中文。
Telegram也遇到同样的问题。
上网搜索过后，原来是安装包的问题。只要把原来的卸载了，重新手工安装（双击或者dpki -i xxx）就可以了。

  • VS Code 通过 link 下载deb包重新安装
  • atom 通过 link 下载deb包重新安装
  • Telegram （参考link）
    • 通过snap源安装
      sudo snap install telegram-desktop
    • 通过ppa源安装
